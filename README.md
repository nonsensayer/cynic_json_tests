# Cynic json tests

This is a public repository that does testing the json form my private standard
library, so it will not work on your computer as I am using my library as a
dependency in here and it is not available. But I don't really want to add test
suites into my dependencies folder and add notices for licenses and such. So i
am just putting this code out there.

If you can, this is how to run:
```
cc -o build build.c
./build all run
```
